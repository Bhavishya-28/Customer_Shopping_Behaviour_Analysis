# 👨‍💻 Customer Shopping Behavior Analysis Project

This project is a complete **end-to-end Data Analytics workflow** where I analyze customer shopping behavior using **Python, SQL Server, and Power BI**.

It demonstrates real-world data analyst skills including **data cleaning, database integration, and dashboard creation**.

---

## 📌 Project Overview

The goal of this project is to transform raw customer data into meaningful business insights.

### 🔹 Key Steps:

* 🐍 **Data Cleaning & Transformation (Python)**

  * Loaded dataset using Pandas
  * Cleaned column names
  * Handled missing values
  * Created new features (age groups, purchase frequency in days)

* 🗄️ **Database Integration (SQL Server)**

  * Connected Jupyter Notebook with SQL Server
  * Created database: `customer_shopping_behaviour`
  * Uploaded cleaned dataset into SQL table (`customer_data`)
  * Queried data using SQL

* 📊 **Data Visualization (Power BI)**

  * Built interactive dashboard
  * Analyzed:

    * Customer segments
    * Purchase trends
    * Revenue patterns

## ▶️ How to Run This Project

1. Clone this repository:

```bash
git clone https://github.com/YOUR_USERNAME/customer-shopping-analysis.git
```

2. Open Jupyter Notebook:

```
Customer_shopping_behaviour.ipynb
```

3. Run all cells:

* Load dataset
* Clean data
* Connect to SQL Server
* Upload data

4. Open SQL Server:

* Check database: `customer_shopping_behaviour`
* Table: `customer_data`

5. Open Power BI file:

```
customer_behaviour_dashboard.pbix
```
